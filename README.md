# OrchNAS

OrchNAS is an energy-aware personalised edge intelligence framework that leverages Neural Architecture Search Service to design device-adaptive models for heterogeneous edge environments.

## Features
- Server-side shared backbone search
- Evolutionary architecture mutation
- Client-specific subnet extraction
- Energy-aware architecture scoring
- Federated training with personalised optimisation

```bash
pip install -r requirements.txt
python train.py
